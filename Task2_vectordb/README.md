# Vector Search API using FASTAPI, ChromaDB and Ollama

- A paragraph is converted into embeddings.
- this embeddings are stored in vector database(chromaDB).
- Users can query dynamically using API.
- The API returns:
   -Best matching text
   -similarity score
   -Embedding vectors

# The task uses:
-**FastAPI** for REST API 
-**chromaDB** for vector storage
-**Ollama** for embeddings
-**Numpy** for vector math
